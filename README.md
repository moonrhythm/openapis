# openapis

Moon Rhythm's Open APIs

## QR Code Generator

`https://qrserver.moonrhythm.io?generate.png?c=<content>`

Parameters
| Parameter | Description | Type | Default |
| --- | --- | --- | --- |
| c | Content | String | |
| l | Recovery Level | Integer (0-3) | 1 |
| s | Image Size | Integer (-10 - 1000) | 256 |
